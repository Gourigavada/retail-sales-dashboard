# 📊 Retail Sales Dashboard (Excel)

## Project Overview

This project is an interactive Excel dashboard created to analyze annual retail sales data. It provides insights into sales trends, customer behavior, order status, sales channels, and regional performance using Pivot Tables, Pivot Charts, and Slicers.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
* Excel Formulas

## Dashboard Features

* Monthly Sales Analysis
* Order vs Sales Analysis
* Sales by Gender
* Order Status Analysis
* Top Performing States
* Sales by Age Group
* Sales by Sales Channel
* Interactive Slicers

## Key Insights

* Women contributed 64% of total sales.
* 92% of orders were successfully delivered.
* Amazon generated the highest sales share.
* Maharashtra recorded the highest sales.
* Adult customers contributed the highest number of purchases.
